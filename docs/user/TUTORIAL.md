# 用户教程（网站使用版）

更新时间: 2026-04-22

本教程仅面向直接使用 Rosetta 网站的用户，不包含部署、密钥配置、运维脚本等内容。

## 1. 使用流程总览

1. 进入「概念管理」准备概念。
2. 进入「智能标注」选择概念并提交文本。
3. 如需生成语料，进入「Corpus Studio」按步骤构建语料库。
4. 查看标注结果、可视化和历史记录。

## 2. 首页功能

1. 快速统计：概念数量、标注历史数量、平均标注长度。
2. 核心功能入口：
- 多模型支持（跳转到智能标注）
- 概念管理
- 智能标注
 - Corpus Studio
3. 最近使用概念与最近标注记录：可快速回看。

## 3. Corpus Studio

`Corpus Studio` 适合从一句话需求出发，逐步生成指定领域 / 体裁 / 语言的语料。

1. 第一步：输入一句话 brief，并补充语言、领域、体裁、受众、文章数量和目标长度。
2. 第二步：系统生成标题候选、策略摘要和 1-2 个样稿方向。
3. 第三步：人工微调标题与策略，并确认进入样稿阶段。
4. 第四步：先生成少量样稿，确认风格和信息密度。
5. 第五步：批量生成完整语料库。
6. 第六步：运行 judge，查看每篇文章的评分、问题和修改建议。

### 3.1 Corpus Studio 输出内容

1. 标题候选列表。
2. 样稿文章。
3. 批量生成后的完整文章集。
4. Judge 评分表与整体总结。

## 4. 概念管理

1. 创建概念：填写名称、分类、提示词和样例。
2. 编辑概念：修改已有概念内容。
3. 删除概念：移除不再需要的概念。
4. 导入概念：支持 JSON 文件导入，支持覆盖或追加。
5. 导出概念：下载当前全部概念 JSON。

### 4.1 样例格式要求

每个 `examples[*]` 需包含 3 个字段：
1. `text`
2. `annotation`
3. `explanation`（必填，不能为空）

## 5. 智能标注

1. 在左侧选择平台、模型和温度参数。
2. 在主区域选择标注概念。
3. 输入待标注文本并点击「开始标注」。
4. 查看结果区输出。

### 5.1 标注格式（V2）

1. 显性标注：`[原文片段]{概念标签}`
2. 隐含义标注：`[!隐含义]{概念标签}`

说明：
1. `[]` 内是被标注内容。
2. `{}` 内是概念标签。
3. 隐含语义必须使用 `!` 前缀。

### 5.2 标注结果查看

1. JSON 结果（默认折叠）。
2. 一键复制完整 JSON。
3. 标注统计：
- 标注片段数
- 标签种类数
- 隐含标注数
4. 标注可视化：高亮片段 + 标签显示。

## 6. 标注历史

1. 最近历史支持展开查看完整内容。
2. 可删除单条历史，或“重新使用此文本”。
3. 支持“下载全部历史”，导出当前会话中的全部标注记录（JSON）。

## 7. 常见使用问题

1. 没有可用模型：这通常是服务端配置问题，请联系管理员。
2. 导入概念失败：根据界面提示修复字段格式，重点检查 `annotation` 和 `explanation`。
3. 页面显示维护提示：说明当前处于调试窗口期，阅读提示后继续使用即可。
