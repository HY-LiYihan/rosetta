# Roadmap

Last updated: 2026-05-08

Near-term priorities:

1. Keep Streamlit as the single official UI foundation.
2. Keep definition validation and optimization visible and auditable.
3. Move LLM calls toward the shared runtime.
4. Preserve Prodigy-compatible JSONL export.
5. Separate optimizable concept definitions from frozen output protocol.
6. Strengthen held-out and PLM baseline experiments before broad claims.
