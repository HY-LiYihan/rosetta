# Annotation JSONL Format

Last updated: 2026-05-08

Rosetta exports annotation data in a Prodigy-compatible JSONL profile. This keeps span offsets, labels, review metadata, model metadata, and run identifiers available for downstream training, evaluation, and audit.

Use this format for PLM/LLM comparison experiments and for moving data into other annotation or training tools.

Canonical Chinese reference: [Annotation JSONL Format](../../developer/ANNOTATION_JSONL_FORMAT.md).
